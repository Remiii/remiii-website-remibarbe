<a href="http://smallerapp.com/" hreflang="en">Smaller</a> est un super minifieur (JS/CSS) pour Mac OSX basé sur <a href="http://developer.yahoo.com/yui/compressor/" hreflang="en">Yahoo YUI Compressor</a>.
